# Ver cambios en Chrome sin volver a ejecutar

1. **Ejecuta una sola vez** en la terminal (desde la raíz del proyecto):
   ```bash
   flutter run -d chrome
   ```
2. **Deja esa terminal abierta** (no la cierres).
3. Cuando cambies código:
   - **Recarga la página en Chrome** (F5 o Ctrl+R) para ver los cambios.
   - O en la terminal donde corre Flutter pulsa **`r`** (hot reload) o **`R`** (hot restart).

No hace falta cerrar la app ni ejecutar `flutter run` de nuevo; con recargar en Google Chrome basta.
